# Bluejay Web Design

## Project Overview

This folder contains everything related to the Bluejay Web Design website project. The goal is to keep the website, backups, documentation, and design assets organized and easy to maintain.

---

## Folder Structure

### website/

The live working website.

This is the only folder where website files are edited.

Examples:

- HTML
- CSS
- JavaScript
- Images
- Fonts
- Documents

---

### backups/

Copies of the website created before major changes.

Each backup folder is dated so previous versions can be restored if necessary.

---

### development-log/

Project documentation.

Contains:

- Development journal
- Project notes
- To-do list
- Design decisions
- Bug tracking
- Reports
- Screenshots

---

### design-assets/

Original design files that are not part of the website itself.

Examples:

- Photoshop (.psd)
- Illustrator (.ai)
- Original photos
- Logos
- Mockups

---

### lighthouse-reports/

Stores Lighthouse reports and performance testing results.

---

### old-college-projects/

Archived college work kept for reference.

These projects are not part of the Bluejay website.

---

## Development Workflow

1. Create a backup before major changes.
2. Make one change at a time.
3. Test locally.
4. Verify desktop and mobile layouts.
5. Run Lighthouse when appropriate.
6. Record important changes in the development log.

---

## Project Goals

- Fast loading
- Responsive design
- Accessible
- SEO friendly
- Easy to maintain
- Professional appearance
- Clean, organized code
