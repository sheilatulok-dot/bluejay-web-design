# Changelog

All notable changes to the Bluejay Web Design project are documented here.

## 2026-07-19

### Project Organization

- Created a professional project folder structure.
- Added a `README.md` describing the project.
- Added a `development-log` folder for project documentation.
- Added folders for notes, reports, and screenshots.
- Organized backups and design assets into dedicated folders.

### Documentation

- Created project documentation system.
- Added TODO list.
- Added Ideas, Decisions, and Bugs notes.
- Began using Visual Studio Code for Markdown documentation.

### Performance Optimization

- Created a baseline Lighthouse report.
- Optimized the website logo.
- Optimized the hero image (first pass).
- Improved Lighthouse Performance score from **77 → 81**.

### Development Workflow

- Established a backup-before-changes workflow.
- Started documenting project decisions and progress.
- Adopted an incremental, one-change-at-a-time development process.
### Responsive Hero Image

- Implemented responsive hero images using `srcset` and `sizes`.
- Mobile devices now receive the optimized 960px hero image.
- Desktop devices receive the original high-resolution hero image.
- Updated the image preload to support responsive image selection.
- Preserved desktop image quality while maintaining mobile performance.
